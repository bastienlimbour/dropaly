# DropZen

DropZen is a mobile-first brain dump app for capturing thoughts quickly and turning them into structured tasks, notes, and list items.

The goal is simple: empty your mind without friction, then let the app help you organize what matters.

## Product Vision

"Drop everything on your mind. DropZen helps structure the rest."

DropZen is not trying to be a full second brain. It is an app designed to help you empty yours quickly.

## MVP Scope

The MVP focuses on:

- Fast text and voice capture.
- AI-assisted creation of tasks.
- AI-assisted addition of items to existing lists.
- Explicit validation before anything is created.
- Inbox triage for tasks without date or project.
- Today and Upcoming views for daily execution.
- Projects, lists, notes, and tasks managed manually.
- Mobile-first web app with PWA support.

## Core Concepts

- **Inbox**: triage area for incomplete items.
- **Tasks**: actionable items with optional date, time, project, and priority.
- **Lists**: simple lists with todo/done items.
- **Notes**: simple text notes linked optionally to a project.
- **Projects**: lightweight containers for tasks, lists, notes, and sub-projects.

## MVP Constraints

- French-only interface.
- Individual account only.
- Online-only.
- No native mobile or desktop app.
- No reminders, calendar, search, sharing, tags, or integrations yet.
